    1° Ano ou modulo

    Qualificação profissional técnica de nivel médio de auxiliar em desenvolveimento de sistemas.

    O auxiliar em desenvolveimento de sistemas é o profissional que desenvolve programas e auxilia na análise de sistemas e modelagem de banco de dados.

    Atribuições responsavéis:

    - Modelar banco de dados

    - Desenvolver sites para web

    - Operar sistemas computacionais

    - Elaborar projetos de sistema de informação

    - Desenvolver elementos gráficos para aplicativos e sites

    - Realizar versionamento no desenvolveimento de programas.

    - Utilizar algoritmos em linguagens de programação.

    2° Ano ou modulo

    Qualificação profissional técnica de nível médio de programador de computadores.

    O programador de computadores é o profissional que desenvlve e documenta projetos de baixa complexidade com banco de dados e múltipla plataformas.

    Atribuições responsavéis:

    - Implementar banco de dados

    - Códificar e depurar programas

    - Realizar gestão de banco de dados

    - Elaborar sistsmas de informação para web

    - Elaborar projetos de aplicativos de aplicativos móveis

    - Atuar de acordo com principios éticos nas relações de trabalho

    - Selecionar linguagens de programação

    3° Ano ou modulo

    Qualificação profissional de técnico em desenvolveimento de sistemas.

    O técnico em desenvolvimento de sistemas é o profissional que analisa e projeta sistemas. Constroi, documenta, realiza testes, e mantém sistemas de informação.

    Atribuições responsavéis:

    - Documentar sistemas da informação

    - Analisar modelos de sistemas embarcados

    - Desenvolver aplicações com microcontroladores

    - Avaliar e selecionar técnicas de teste de software

    - Elaborar modelo de negócios para uma empresa de software

    - Desenvolver sistemas, implementando rotina de seguraça de dados

    - Avaliar as fontes e recursos necessários para o desenvolvimento de projetos
